# Workflow: Learn Preference

1. Analyze the last 3-5 messages in the conversation to identify the user's specific preference (e.g., formatting, language choice, or technical constraints).
2. Summarize this preference into a clear, bulleted Markdown rule.
3. Use the **knowledge-learner** skill to save this rule into the workspace.
4. Inform the user that the agent will now follow this rule automatically.
