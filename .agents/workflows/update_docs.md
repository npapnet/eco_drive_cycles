# Workflow: Learn Preference

1. Analyze the package folder `<reporoot>/src/`.
2. based on the relevant state of the src update the documents in 
   a) `<reporoot>/docs/design` 
   b) `<reporoot>/CHANGELOG.md` (put done items at the end and add date of the addition).
   c) `<reporoot>/README.md`.
   d) `<reporoot>/TODOS.md`.

3. Ask the viewer to review the changes.
